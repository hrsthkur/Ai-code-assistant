# AI Code Assistant

A small web application that analyzes code, explains what it does, points out potential issues, and suggests improvements. The app includes a code editor and a chat-style interface to ask follow-up questions.

## Features
- Paste or write code directly in the editor
- Get explanation of the code logic
- Detect possible bugs or risky expressions
- Receive suggestions for improvement
- Chat interface to ask additional questions

## Tech Stack

**Frontend**
- React  
- Tailwind CSS  

**Backend**
- Node.js  
- Express  

**AI Model**
- Groq LLaMA API  

**Deployment**
- Frontend: Vercel  
- Backend: Render
